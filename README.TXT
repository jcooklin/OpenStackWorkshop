In these labs users will be introduced to nova command line interface and pyhton nova client API to interact with the cloud and be able to launch instances



Lab 1 : Managing Openstack through nova CLI (Command Line Interface) and using Dashboard 
       a. Users will be introduced to nova CLI. We will explore some of the system, access & security  and instance level commands
       b. Using different nova commands users will be able to check services available to interact with cloud, be able to launch instances in the cloud, create security keypairs and login into the instance created. 



Lab 2 : Programming OpenStack, Its the API: 
	a. users will be introduced to differnt python API available to interact with the cloud. we will explore some of the system and instance level API
        b. Similar to what we have done in Lab 1, users will be able to use python nova client APIs to check system level information like the available images, flavors etc to launch instances.
